SignAR – AI & AR Based Sign Language Avatar System
Overview

SignAR is an AI-powered Sign Language Recognition and AR/VR Avatar System developed during my Software Development & AR/VR Internship at Indian Institute of Technology Hyderabad under the PMKVY–NSDC program.

The system recognizes sign language gestures in real time using Computer Vision and Deep Learning techniques, converts predictions into meaningful actions through a FastAPI backend, and animates a 3D avatar in Unity3D to provide an interactive communication experience.

Features
Real-time sign language recognition
Hand landmark detection using MediaPipe
Gesture classification using PyTorch
Image processing with OpenCV
FastAPI-based prediction service
Unity3D avatar animation integration
AR/VR workflow validation
REST API testing with Postman
Functional and usability testing
Agile development methodology
